# 🚀 adi-chaitailwind

A lightweight utility-first CSS engine inspired by Tailwind CSS — built from scratch using pure JavaScript.

---

## ✨ Features

- ⚡ Lightweight and fast
- 🎯 Utility-first approach
- 🧠 Custom class parser
- 🛠️ Built using vanilla JavaScript
- 📦 Easy to integrate

---

## 📥 Installation

```bash
npm install adi-chaitailwind

```
